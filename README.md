# AI Tools Hub - Admin Panel

- ✅ إضافة أدوات جديدة
- ✏️ تعديل الأدوات الموجودة
- 🗑️ حذف الأدوات
- 🔍 بحث وفلترة

### 📊 أقسام متعددة
- 🌐 أدوات أونلاين (Online Tools)
- 💻 تطبيقات سطح المكتب (Desktop Apps)

### 🔄 تكامل GitHub
- 💾 حفظ تلقائي على GitHub
- 🔐 استخدام GitHub كقاعدة بيانات آمنة
- 📝 تتبع جميع التغييرات (Commit History)
- 🌍 إمكانية الوصول من أي مكان

### 🌐 دعم اللغات
- 🇬🇧 الإنجليزية
- 🇸🇦 العربية (RTL Support)

### 📤 استيراد/تصدير

- 📤 تصدير البيانات إلى JSON
- 🔄 نسخ احتياطي سهل

## 🚀 الإعداد والتثبيت

### 1. إنشاء Repository على GitHub

1. اذهب إلى [GitHub](https://github.com)
2. أنشئ Repository جديد (مثلاً: `ai-tools-database`)
3. اختر Public أو Private حسب رغبتك

### 2. الحصول على Personal Access Token

1. اذهب إلى GitHub → Settings → Developer settings
2. اختر **Personal Access Tokens** → **Tokens (classic)**
3. اضغط على **Generate new token (classic)**
4. أعط التوكن اسماً (مثلاً: AI Tools Admin)
5. اختر الصلاحيات التالية:
   - ✅ `repo` (Full control of private repositories)
6. اضغط **Generate token**
7. **انسخ التوكن واحفظه** (لن تتمكن من رؤيته مرة أخرى!)

### 3. إعداد لوحة التحكم

1. افتح ملف `admin.html` في المتصفح
2. سيظهر نافذة إعدادات GitHub تلقائياً
3. أدخل المعلومات التالية:
   - **GitHub Repository Owner**: اسم المستخدم الخاص بك
   - **GitHub Repository Name**: اسم الـ Repository (مثلاً: `ai-tools-database`)
   - **GitHub Personal Access Token**: التوكن الذي حصلت عليه
4. اضغط **Connect**

### 4. البدء في استخدام لوحة التحكم

الآن يمكنك:
- ✨ إضافة أدوات جديدة
- ✏️ تعديل الأدوات
- 🗑️ حذف الأدوات
- 🔄 سيتم حفظ كل شيء تلقائياً على GitHub!

## 🎨 استخدام لوحة التحكم

### إضافة أداة جديدة

1. اختر القسم من القائمة الجانبية
2. اضغط على **Add New Tool**
3. املأ التفاصيل:
   - **اسم الأداة**: مثل ChatGPT
   - **الوصف** (إنجليزي وعربي)
   - **رابط الموقع**: URL الأداة
   - **القسم**: Online/Desktop/Mobile/Extensions
   - **التصنيف**: للأدوات الأونلاين فقط
   - **التسعير**: Free/Freemium/Paid
   - **الشارة**: Popular/New/Premium/Trending
   - **أيقونة**: FontAwesome class (مثل: `fas fa-robot`)
   - **خلفية الأيقونة**: CSS gradient
   - **المميزات**: بالفاصلة (إنجليزي وعربي)
   - **المنصات**: للتطبيقات فقط
4. اضغط **Save Tool**

### تعديل أداة

1. اختر القسم
2. اضغط على زر ✏️ بجانب الأداة
3. عدّل ما تريد
4. اضغط **Save Tool**

### حذف أداة

1. اختر القسم
2. اضغط على زر 🗑️ بجانب الأداة
3. أكد الحذف

## 🛡️ الأمان

### ⚠️ مهم جداً:
- **لا تشارك GitHub Personal Access Token** مع أحد

## 🐛 حل المشاكل

### GitHub API Error 401
- تأكد من صحة التوكن
- تحقق من صلاحيات التوكن (`repo`)

### GitHub API Error 404
- تأكد من اسم المستخدم و Repository
- تأكد أن Repository موجود

### لا يتم الحفظ
- افتح Console في المتصفح (F12)
- اقرأ رسائل الخطأ
- تأكد من اتصالك بالإنترنت

### البيانات لا تظهر
- تأكد من وجود ملف `data/tools.json` في Repository
- جرب الضغط على زر المزامنة 🔄
